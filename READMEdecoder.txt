Archivo README para el funcionamiento del decodificador en python.

Pasos para usarlo:

1) Colocar el archivo instrucciones.py en una carpeta.

2)Introducir en la misma carpeta un archivo de texto de nombre "programa.txt". En él deberán estar todas las instrucciones necesarias.

3) Las instrucciones pueden ser escritas con o sin ','
   Es decir:
   Add $7, $3, $5
   Add $7 $3 $5
   Estos son los formatos válidos

4)Ejecutar el archivo de python.

5)Se creará un archivo un nuevo archivo de texto de nombre Mem_Inst.txt, el cual se podrá precargar a la memoria de instrucciones para su funcionamiento.
